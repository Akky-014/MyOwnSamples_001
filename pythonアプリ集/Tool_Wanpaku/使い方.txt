# 手順
①pythonを自分のPCにインストール
https://www.python.org/downloads/

②インストール後、コマンドプロンプトから以下コマンドを実行
pip install pandas 
pip import glob

③C直下にこのフォルダをまるごと入れる

④参加者リストaとbをcsv方式にしてきてダウンロードしたあと、_csvというフォルダに２つとも入れる

⑤あとはバッチファイルを実行するのみ！
